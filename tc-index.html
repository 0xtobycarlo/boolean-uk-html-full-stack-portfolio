Portfolio

Solidity Lottery

This mini-project is a lottery system built on the Ethereum blockchain, using its native programming language Solidity. It is a mechanism that could be interpreted and used by NFT projects as a way of incentivising whitelist & airdrop participation (this comes with increased social scope) in return not only for first-access to the main drop of a project, but also for a chance at winning the left-over profits from the project's sales. A potential reward for being an early adopter essentially.

Languages Used: Solidity.
